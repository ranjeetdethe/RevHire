# RevHire Frontend (Angular)

This is the Angular 17 frontend for the RevHire job portal.

## Run (local)

```bash
npm install
npm start
```

Frontend runs at:
- `http://localhost:4200`

Backend API base URL is configured in:
- `src/environments/environment.ts` (default: `http://localhost:8080/api/v1`)

## Build

```bash
npm run build
```
